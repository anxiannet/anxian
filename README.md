# 暗线（ANXIAN）

一个人的调查游戏。

所有人的城市传说。

## 项目简介

《暗线》是一款都市调查题材 AI 剧情游戏。

玩家将进入虚构城市：

白港市（White Harbor）

通过调查异常事件、收集线索、建立人脉与解锁档案，逐步接触隐藏在城市背后的「暗线网络」。

## 核心特色

- 都市异闻与悬疑调查
- CoC Lite 轻量化体验
- AI 动态剧情
- 单人即可游玩
- 无限案件生成
- 第一视角调查
- 图片、语音与聊天记录证据系统

## 世界观

白港市是一座繁荣的现代海港城市。

这里每天都有新的故事发生。

其中一些故事会登上新闻。

另一些故事，则会被记录在暗线之中。
