# 《暗线（ANXIAN）》视觉与场景演出规范 v1.0

## 核心原则

玩家不是在看故事，而是在经历故事。

- 画面优先
- 情绪优先
- 行动优先
- 调查后置

禁止：长篇说明、世界观灌输、小说式开场。

## 项目气质

白港市是一座现代、国际化、高度数字化的海港城市。

参考：
- 《开端》
- 《隐秘的角落》
- 《白夜追凶》
- 《消失的她》

不参考：
- 废土
- 末日
- 鬼屋
- 恐怖生存游戏

核心定位：现实中的异常事件。

## 美术风格

关键词：

- 写实
- 电影感
- 现代都市
- 第一视角
- 手机阅读友好
- 高细节
- 信息优先

## 场景构图

统一竖屏 9:16。

布局：
- 上方20%：程序动态演出区域
- 中部55%：核心场景区域
- 下方25%：UI与选择区域

图片本身不绘制按钮与系统UI。

## 第一视角规范

玩家必须存在于场景中。

允许：
- 双手
- 双腿
- 手机
- 方向盘
- 电脑
- 猫眼视角

禁止：
- 上帝视角
- 第三人称
- 地图视角

## 白港市场景规范

允许：
- 公寓
- 写字楼
- 地铁
- 商场
- 停车场
- 医院
- 学校
- 海港
- 咖啡馆

禁止：
- 破木门
- 废弃建筑
- 末日废土
- 恐怖屋

## 光影原则

信息优先于氛围。

重要信息必须清晰可见：
- 手机内容
- 证据
- 异常物
- 关键入口

即使在白天手机屏幕上也必须清晰辨认。

## 场景图职责

场景图只负责展示：

发生了什么。

程序负责展示：

发生什么变化。

## 场景信息层级

1. 场景
2. 环境拟声
3. 人物对白
4. 手机消息
5. 玩家选择

## 拟声表现

采用漫画式空间拟声。

例如：
- 砰！！
- 咚！！
- 嘎吱——

拟声出现在事件发生的位置。

## 对白表现

采用空间对白。

例如：
- 门外：开门！
- 走廊深处：后面的人上！
- 远处：快点！

禁止传统滚动弹幕。

## 手机信息规范

短。

少。

有悬念。

示例：
- 不要开门。
- 他们找到你了。
- 不要让他们看到手机。

## 场景节奏

场景
→ 动态演出
→ 新信息
→ 玩家选择
→ 下一场景

## 选择规范

优先行动，而非分析。

正确：
- 查看猫眼
- 顶住房门
- 躲进厕所

错误：
- 分析现场
- 推理原因

## 首页标准

玩家进入后3秒内必须知道：

- 我在哪
- 发生了什么
- 我要做什么

首页目标不是介绍世界观。

首页目标是让玩家产生：

“卧槽，我好像出事了。”

## 首页标准镜头（当前版本）

现代青年公寓。

暴雨夜。

第一视角。

智能防盗门关闭。

门底透光。

门外影子不断经过。

猫眼可见。

手机显示：

“不要开门。”

“他们找到你了。”

随后由程序控制：

- 手机震动
- 门把手抖动
- 猫眼变暗
- 拟声出现
- 选择弹出

形成完整沉浸式短剧体验。
