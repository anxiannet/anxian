# 《暗线》案件生成指令

目标：

生成适合《暗线》的短时长调查案件。

案件必须满足：

- 简单直白
- 矛盾冲突强烈
- 有悬念
- 有反差
- 有反转
- 有惊喜

案件结构：

事件 → 调查 → 线索 → 矛盾 → 推理 → 真相 → 档案

输出内容必须包含：

1. 案件标题
2. 案件等级
3. 案件简介
4. 场景列表
5. NPC列表
6. 线索列表
7. 真相
8. 隐藏真相
9. 档案记录
10. 场景图片提示词

场景描述采用第一人称视角。

每个场景必须提供用于AI生成图片的提示词。

AI负责：

- 对白
- 氛围
- 场景描述

系统负责：

- 真相
- 逻辑
- 关键线索
- 结局判定

原则：

玩家看到的是剧情。

后台运行的是结构化案件引擎。
