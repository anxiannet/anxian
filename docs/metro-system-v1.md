# 白港轨道交通系统设定 V1.0

## 项目

白港轨道交通（White Harbor Metro）

简称：WHM

---

## 系统简介

白港轨道交通（WHM）是白港市最大的公共交通系统。

承担全市约68%的公共交通客运量。

覆盖：

- 白港区
- 海湾区
- 东港区
- 西港区
- 北岸区
- 大学城
- 工业带

全网长度：116.4公里

车站数量：84座

换乘站：6座

日均客流：286万人次

---

# 1号线（红线）

North-South Line

北岸区 → 白港区 → 海湾区

主要站点：

- 北岸花园
- 北岸公园
- 白港医院
- 中央车站
- 金融中心
- 海湾公园
- 海湾终点站

特点：客流量最高

---

# 2号线（绿线）

East-West Line

西港区 → 白港区 → 东港区

主要站点：

- 西港码头
- 西港市场
- 市政府
- 中央车站
- 白港广场
- 物流中心
- 东港码头

特点：商业活动最活跃

---

# 3号线（紫线）

Bay Industrial Line

海湾区 → 白港区 → 工业带

主要站点：

- 海湾度假区
- 创新中心
- 海湾公园
- 白港广场
- 科技园
- 东方工业园
- 工业带北区

特点：科技与工业走廊

---

# 4号线（黄线）

Circle Line

海湾区 → 东港区 → 工业带 → 北岸区 → 大学城 → 西港区 → 海湾区

特点：环线
