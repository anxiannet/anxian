# 《暗线（ANXIAN）》项目总指令

https://github.com/anxiannet/anxian.git

开始工作前必须阅读项目文档。

---

# 项目定位

《暗线》是一款：

AI剧情游戏
+
都市调查
+
CoC Lite
+
BitLife式人生模拟
+
悬疑短剧

---

项目口号：

一个人的调查游戏。

所有人的城市传说。

---

# 核心原则

玩家打开游戏后：

1分钟进入事件

5分钟卷入谜团

10分钟完成一次调查体验

---

禁止：

复杂TRPG规则

复杂建卡

长篇世界观

大量说明文本

强制多人玩法

数值刷怪

MMORPG玩法

---

允许：

单人调查

聊天式剧情

图片证据

语音证据

监控录像

档案系统

无限案件

---
