# 《暗线（ANXIAN）》案件镜头规范 v1.0

## 核心原则

案件不是小说。

案件不是剧本。

案件是互动短剧。

玩家体验：

- 3秒进入危险
- 30秒进入事件
- 3分钟发现矛盾
- 5分钟开始调查
- 10分钟获得真相

## 案件结构

标准案件：7~12个镜头。

每个镜头只传达一个核心信息。

禁止一个镜头塞入大量信息。

## 标准镜头流程

### 镜头1：危险

目的：立即卷入事件。

允许：
- 有人撞门
- 电话求救
- 监控异常
- 电梯停电
- 公交失控

禁止：
- 世界观介绍
- 人物介绍
- 背景设定

### 镜头2：升级

危险扩大。

例如：
- 更多人出现
- 第二条短信
- 新异常发生

### 镜头3：发现

出现第一个关键线索。

例如：
- 照片
- 监控
- 录音
- 物证

### 镜头4：选择

第一次主动行动。

正确：
- 查看猫眼
- 接电话
- 打开录像

错误：
- 分析情况
- 推理原因

### 镜头5：调查

获取更多信息。

### 镜头6：矛盾

出现事实冲突。

例如：
监控显示玩家在现场。

但玩家确信自己没去过。

### 镜头7：反转

推翻玩家原有认知。

### 镜头8：真相

解释表层谜团。

必须简单直接。

### 镜头9：隐藏真相

连接暗线世界观。

埋下后续事件。

## 场景图规则

一镜头一张图。

图片负责展示事件。

程序负责展示变化。

禁止：
- 长漫画
- 大量文字
- 复杂说明

## 程序动态演出

允许：
- 拟声
- 空间对白
- 手机震动
- 灯光变化
- 倒计时
- 门锁变化

## 空间对白规范

正确：
- 开门！
- 快点！
- 他就在里面！

禁止长篇对白。

一句话一条。

## 手机信息规范

每条不超过15字。

例如：
- 不要开门。
- 他们找到你了。
- 不要让他们看到手机。

## 选择规范

每次2~4个选项。

必须是行动。

正确：
- 查看猫眼
- 躲进厕所
- 抢手机

错误：
- 分析现场
- 推理原因

## 节奏公式

危险
→ 选择
→ 发现
→ 调查
→ 矛盾
→ 反转
→ 真相
→ 隐藏真相

## 案件结束标准

玩家必须知道：

- 发生了什么
- 为什么发生
- 谁参与了

同时留下新的疑问。

## 暗线黄金公式

危险吸引玩家。

矛盾驱动调查。

反转制造惊喜。

隐藏真相连接世界观。

案件结束后，玩家应产生：

“原来如此。”

然后：

“等等……那这个又是什么？”
